# Worktrees

## What it is

A set of Copilot Chat prompts for managing git worktrees — creating, listing, switching between, and deleting them.

## Who it's for

Developers who work on multiple branches or tasks in parallel and want to avoid the friction of stashing, committing, or losing context when switching workstreams.

## What problem it solves

Switching branches mid-task interrupts your flow and risks losing work-in-progress. Git worktrees let you check out multiple branches simultaneously in separate directories, so you can jump between workstreams instantly. These prompts help you set up and manage that workflow without memorizing the commands.
