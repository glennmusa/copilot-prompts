# Code Review

## What it is

A set of Copilot Chat prompts that perform a thorough code review on a file or selection, flagging bugs, security issues, style violations, and opportunities for improvement.

## Who it's for

Developers who want a fast, automated first-pass review before opening a pull request, or anyone who wants a second opinion on code they've written or inherited.

## What problem it solves

Manual code review is time-consuming and easy to skip under deadline pressure. These prompts give you instant, consistent feedback so you can catch problems early — without waiting on a teammate.
