# Write Docs

## What it is

A set of Copilot Chat prompts that generate documentation for your code — including inline comments, docstrings, README files, API references, and usage guides.

## Who it's for

Developers who know their code inside and out but struggle to write clear documentation, or teams whose documentation consistently lags behind the code.

## What problem it solves

Undocumented code is a liability: it slows onboarding, increases bus factor, and makes maintenance harder. These prompts produce accurate, human-readable docs directly from the source so the gap between code and documentation closes automatically.
