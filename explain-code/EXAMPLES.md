# Explain Code Examples

## Plain-language summary

```
Explain the selected code in plain language. What does it do, and how does it do it? Assume I'm a competent developer but unfamiliar with this codebase.
```

## Step-by-step walkthrough

```
Walk me through the selected code line by line. Explain what each section does and why it might have been written that way.
```

## Identify gotchas and surprises

```
Read the selected code and point out anything surprising, non-obvious, or likely to cause bugs. What would a developer need to know before modifying this code?
```

## Explain an algorithm

```
Explain the algorithm used in the selected code. What is its time and space complexity? Are there trade-offs or alternative approaches I should know about?
```

## Summarize a file

```
Summarize #file. What is its purpose, what are its main components, and how does it fit into the broader codebase?
```

## Explain a dependency

```
The selected code uses [library/framework]. Explain what it is, why it's being used here, and what the key concepts I need to understand are.
```
