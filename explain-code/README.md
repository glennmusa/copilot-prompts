# Explain Code

## What it is

A set of Copilot Chat prompts that produce plain-language explanations of complex, unfamiliar, or legacy code — including what it does, why it was written that way, and any hidden gotchas.

## Who it's for

Developers onboarding to a new codebase, engineers debugging code they didn't write, or anyone staring at a function and thinking "what does this even do?"

## What problem it solves

Understanding unfamiliar code is one of the most time-consuming parts of software development. These prompts compress hours of spelunking into a concise summary so you can get up to speed fast.
