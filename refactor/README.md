# Refactor

## What it is

A set of Copilot Chat prompts that improve the structure, readability, and performance of existing code without changing its observable behavior — applying patterns like extraction, simplification, and deduplication.

## Who it's for

Developers dealing with technical debt, anyone who needs to make code more maintainable before extending it, or engineers doing a cleanup sprint.

## What problem it solves

Code rots over time. Ad-hoc additions create duplication, long functions, and unclear logic. These prompts guide targeted, safe refactoring so the codebase stays healthy without risky rewrites.
