# Write Tests Examples

## Generate unit tests

```
Write unit tests for the selected function using the testing framework already used in this project. Cover the happy path, edge cases, and error conditions.
```

## Generate tests for a whole file

```
Write a complete test file for #file. Use the testing framework and conventions already present in this project. Include tests for every exported function.
```

## Add missing edge case tests

```
Look at the selected function and the existing tests for it. What cases are not covered? Write additional tests to fill the gaps.
```

## Test a React component

```
Write tests for the selected React component using React Testing Library. Cover rendering, user interactions, and any conditional display logic.
```

## Generate integration tests

```
Write integration tests for the selected code that test it end-to-end, including its interactions with dependencies. Mock only external services, not internal modules.
```

## Test error handling

```
Write tests specifically for the error handling paths in the selected code. Ensure every thrown error and rejected promise is covered.
```
