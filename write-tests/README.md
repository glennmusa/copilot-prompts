# Write Tests

## What it is

A set of Copilot Chat prompts that generate unit tests, integration tests, and edge-case coverage for existing code using the testing framework already present in your project.

## Who it's for

Developers who want to increase test coverage quickly, or anyone who finds writing boilerplate test code repetitive and tedious.

## What problem it solves

Writing tests for existing code is often skipped because it feels like busywork. These prompts generate comprehensive, project-aware test suites in seconds so there's no excuse not to have coverage.
